# 実装仕様 v2 - 02 ルーティング決定仕様

## 優先順位
1. 明示コマンド
2. ルール辞書（強証拠）
3. 分類器（1入力1回）
4. 安全側フォールバック

## 重要ルール
- `/local` 中の `/code` は Router で拒否
- 分類器の出力は JSON のみ
- CODE は高閾値（`confidence >= 0.8`）かつ強証拠必須

## 追加管理
- 分類器 system prompt は1枚固定（カテゴリ追加時のみ更新）
