# 実装仕様 v2 - 05 ワーカー仕様

## 共通
- 出力は JSON のみ
- 失敗時は `needs_next_loop=false`
- `next_actions` / `questions_for_user` は最大3

## 各ワーカー
- PLAN / ANALYZE / OPS / RESEARCH: 推論LLM
- CODE: クラウドLLM（`local_only` 時は禁止）

## route不一致時
- `fit=false` と `suggested_route` で再ルート提案可能
