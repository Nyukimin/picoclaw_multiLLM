最終更新: 2026-02-24
対象: `docs` 配下の `.md` 全40ファイル（+3: CHAT_PERSONA設計.md, Coder3_Claude_API仕様.md, Coder3承認フロー実装プラン.md）

## 分類ルール
- 主分類は「正本仕様 / 分割仕様 / 旧版アーカイブ / 監査メモ / LLM運用設計 / 実装ガイド / コミュニティ資料」で分ける。
- 実施区分タグは次を使用する。  
  - `coding`: 実装時に直接参照する仕様  
  - `ops`: 運用時に参照する手順  
  - `prompt`: プロンプト資産として使う  
  - `reference`: 背景・比較・監査の参照用  
  - `archive`: 履歴保存用
